# <a name="microsoft-graph-training-module---build-microsoft-teams-apps-with-the-microsoft-graph-sdk"></a>Módulo de aprendizaje de Microsoft Graph: crear Microsoft Teams aplicaciones con el SDK Graph Microsoft

Este módulo le presentará cómo trabajar con el SDK de Microsoft Graph para obtener acceso a los datos de Microsoft 365 mediante la creación de una aplicación Microsoft Teams mediante ASP.NET Core.

## <a name="lab---build-microsoft-teams-apps-with-the-microsoft-graph-sdk"></a>Laboratorio: crear Microsoft Teams aplicaciones con el SDK de Microsoft Graph

En este laboratorio, creará una aplicación de consola mediante la Biblioteca de autenticación de Microsoft (MSAL) para obtener acceso a los datos en Office 365 mediante microsoft Graph.
